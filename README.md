## 🛠️ 掲載ツール

### Monitora
Error Management Tool - Unity エディタのエラー管理を劇的に改善する高機能パネル

- 時刻別ログのグループ化
- 強力な検索機能（正規表現対応）
- チェックボックスでログを選択してコピー
- 複数形式のエクスポート（TXT/CSV/Markdown）
- エラー発生時の自動パネル表示
- カスタマイズ可能なUI

**システム要件:**
- Unity 2021.3 以降（推奨）
- Unity Asset Store 提出基準に準拠

> [!WARNING]
> Unity 2019.x / 2020.x での動作は保証されていません。Unity 2021.3 LTS 以降のご利用を強く推奨します。
