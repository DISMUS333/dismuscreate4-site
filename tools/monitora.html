Monitora

Error Management Tool - Unity エディタのエラー管理を劇的に改善

▼ Overview ▼

通常のUnity Consoleでは実現できない、ログの時刻別グループ化、強力な検索機能、チェックボックスでの選択コピー、 柔軟なエクスポート機能を提供し、大量のログを整理された状態で確認・分析できます。

ChatGPTやClaude等のAIアシスタントとの連携に最適化されており、エラーログを瞬時に共有して効率的なデバッグ作業を実現します。

使用方法

  1. Unity エディタのメニューから Window → Monitora を選択
  2. チェックボックスでログを選択、検索バーでフィルタリング
  3. ツールバーの「時刻でグループ化」ボタンをクリック
  4. Export TXT ボタンから形式を選択してエクスポート

◈ Screenshots ◈

グループ化で整理

時刻別に自動グループ化することで、99+のログもワンクリックで折りたたみ・展開できます。カオスを瞬時に整理。

高度なフィルタリング

エラー、警告の個別絞り込み、正規表現対応の検索機能、ページネーション機能で大量のログを効率的に管理できます。

柔軟なエクスポート

TXT、CSV、Markdown形式でログをエクスポート可能。AIアシスタントへの共有にも最適化されています。

高度なカスタマイズ

通知設定、自動オープン、UIカラー、フォントサイズなど、ワークフローに合わせて細かく設定できます。
