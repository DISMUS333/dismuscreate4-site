Error Clipboard Panel

Unity エディタのエラー管理を劇的に改善

▼ Overview ▼

通常のUnity Consoleでは実現できない、ログの時刻別グループ化、強力な検索機能、 柔軟なエクスポート機能を提供し、大量のログを整理された状態で確認・分析できます。

◆ Installation ◆

手動インストール

STEP 1: リポジトリをクローン
git clone https://github.com/DISMUS333/dismuscreate4-site.git
STEP 2: ファイルをコピー

ErrorClipboardPanel.cs を Unity プロジェクトの Assets/Editor フォルダにコピー

※ Editorフォルダが存在しない場合は作成してください

STEP 3: 自動コンパイル

Unity エディタを開くと自動的にスクリプトがコンパイルされます

使用方法

  1. Unity エディタのメニューから Window → Error Clipboard Panel を選択
  2. チェックボックスでログを選択、検索バーでフィルタリング
  3. ツールバーの「時刻でグループ化」ボタンをクリック
  4. Export TXT ボタンから形式を選択してエクスポート
